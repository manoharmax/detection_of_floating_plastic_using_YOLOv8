# detection_of_floating_plastic_using_YOLOv8
## Project Description

This project focuses on detecting floating plastic waste on water surfaces using the YOLOv8 deep learning model. 
The system aims to support environmental monitoring by identifying plastic debris in real time from images or video streams.

## Technologies Used

- Python
- YOLOv8
- OpenCV
- NumPy
- Machine Learning
- Computer Vision
